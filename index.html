Illustration @Container Queries

Cette page démontre l'utilisation des Container Queries, une nouvelle fonctionnalité en css.Les container Queries (reqêtes de containers) vous permettent d’appliquer des styles à un élément en fonction de la taille du conteneur de l’élément. Si, par exemple, un conteneur dispose de moins d’espace disponible dans le contexte environnant, vous pouvez masquer certains éléments ou utiliser des polices plus petites.

Ils sont une alternative aux médias Queries, qui appliquent des styles aux éléments en fonction de la taille de la fenêtre d’affichage ou d’autres caractéristiques de l’appareil.

Pour utiliser des requêtes de conteneur, vous devez déclarer un contexte d’inclusion sur un élément afin que le navigateur sache que vous souhaiterez peut-être interroger les dimensions de ce conteneur ultérieurement.


Dans l'exemple suivant : nous utilisons des cartes pour illustrer le fonctionnement des container Queries. Vous avez la possibilité d'ajouter des cartes ou d'en supprimer. Et vous pouvez réduire ou augmenter la taille de la fenêtre pour observer le comportement des cartes dont la disposition et la taille change en fonction de la taille de la fenêtre et aussi du nombre de cartes.

Extraits de code

Ci-dessous un extrait de code pour avoir une idée de leur utilisation :

CSS (Container Queries):

            
                @container (min-width: 600px) {
                    :host {
                      max-width: 1200px;
                    }
                }
                  
                @container (min-width: 900px) {
                  :host {
                    max-width: 1600px;
                  }
                }

Explication:

Ces requêtes de conteneur ajustent la largeur maximale de l’élément personnalisé en fonction de la largeur du conteneur:

Si la largeur du conteneur est égale ou supérieure à 600 pixels, la largeur maximale de l’élément personnalisé est définie sur 1200 pixels.

Si la largeur du conteneur est supérieure ou égale à 900 pixels, la largeur maximale de l’élément personnalisé est ajustée à 1600 pixels.
