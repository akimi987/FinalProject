Fonctions trigonométriques en css

Descriptif de la fonctonnalité

En CSS, il est possible d'écrire des expressions mathématiques. À la base se trouve la fonction calc() pour effectuer des calculs, mais vous avez probablement déjà entendu parler de min(), max() et clamp(). Dans Chrome 111, ces fonctions trigonométriques sin(), cos(), tan(), asin(), acos(), atan() et atan2() sont associées. Ces fonctions sont définies dans le module Valeurs et unités de niveau 4 du CSS et sont disponibles dans tous les navigateurs.

  • Edge
  • Firefox
  • Safari
